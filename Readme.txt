Steps to run the Project:
1. Unzip the folder
2. Extract the folder onto your desktop.
3. Using Vscode open the folder.
4. Use (cntrl+~) or open a new terminal.
5. Give the following commands.
6. cd backend.
7. npm install.
8. cd ..
9. cd client.
10. npm install.
11. cd ..
12. npm install.
13. npm run dev.
